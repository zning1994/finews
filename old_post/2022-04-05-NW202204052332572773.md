 <p>据 Neowin 报道，Azure 虚拟机现在将能够在基于 ARM 的处理器上运行。微软已经确认，ARM CPU，特别是安培计算公司(Ampere)开发的 CPU，现在将为多个虚拟机提供支持，包括 Web 服务器、开源数据库、游戏服务器、媒体服务器等。</p>
 <p>微软正在积极地让 ARM CPU 参与其生态系统中。在接纳了几乎所有流行的 Linux 发行版作为虚拟机的操作系统之后，Azure 虚拟机平台现在正从硬件方面欢迎 ARM 架构。该 Windows 操作系统制造商已经宣布在 Azure 虚拟机上提供 ARM 支持的“预览”版。该预览版最初在美国西部 2 区、美国中西部和欧洲西部 Azure 地区提供，安培公司证实了这一点。</p>
 <p>安培计算公司是一家专注于服务器级处理器的初创公司，去年曾宣布已签约美国微软和中国腾讯控股作为主要客户。Azure 主机操作系统和 Windows 操作系统平台的 PM 总监 Hari Pulapaka 在推特上确认了预览版的推出。</p>
 <p>“Ampere on Azure with Windows as the Root Host OS”表明，Windows 将是在安培计算机的 ARM CPU 上运行的基本操作系统。但目前预览版中的虚拟机支持 Ubuntu、CentOS 以及 Windows 11 专业版和企业版，Windows 11 on ARM.此外，用于 Azure 的 Ampere Altra ARM CPU 应该很快会支持更多的操作系统，如 Red Hat Enterprise Linux、SUSE Linux Enterprise Server、Debian、AlmaLinux 和 Flatcar。</p>
 <p>微软表示，采用 Ampere Altra ARM 处理器的 Azure 虚拟机在扩展工作负载方面的性价比将比同类的基于 x86 虚拟机高 50% 以上。ARM CPU 可能很快就会运行一些常见的虚拟机，包括 Web 服务器、应用服务器、开源数据库、游戏服务器、媒体服务器等。</p>
 <p>新的虚拟机可以配置多达 64 个 vCPU，并且可以灵活地选择每个 vCPU 内存配置为 2GiB、4GiB 和 8GiB 的虚拟机大小。他们将拥有高达 40Gbps 的网络。用户还将能够选择高性能的本地固态硬盘存储。Dpsv5 和 Epsv5 Azure 虚拟机系列将运行在基于 Ampere Altra ARM 的处理器上，工作频率最高可达 3.0GHz。</p><p class="em_media">（文章来源：IT之家）</p>
