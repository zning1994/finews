<p>近日，在容声WILL SPACE年度新品发布会上，有一段冰箱容积展示的对比引人关注。在预先摆放好食材的一台容声新品WILL SPACE超空间冰箱和一台普通冰箱中依次拿取相同食材，最后可以看见，普通冰箱中已经空了，will space超空间还剩下一些食材，分别是四条羊腿、两扇羊排和一个羊头。will space超空间冰箱神奇的地方，就是占地一样大，近乎多装了一只羊。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;</p><p>国内外的冰箱厂商都在追求小空间里的大容量，这方面日本厂商做的比较好，他们在700毫米宽度以内的冰箱占地约0.47平方米左右。冰箱的容积从330L到470L不等，最高做到了471L。但这个记录已被追求极致的容声打破，占地0.469平，相比之前行业普遍的400多升容积，容声做到了502升。容声，再次在容积率上站了行业颠峰。全行业700毫米宽最大容积的冰箱，由容声创造。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;</p><p>容声通过“宇航级复合绝热材料”+全球领先的背部四孔发泡工艺+全球首条得到联合国多边基金认证并资助的行业示范发泡生产线等新材料，新技术，新工艺的组合式创新，使will space相较于传统产品保温性能提升10倍的同时，冰箱的壁厚也成功减薄44%，内部空间增大20%以上。这正是WILL SPACE的空间魔法。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;</p><p>2019年，容声WILL横空出世。持续生长、极致养鲜的产品理念，让WILL冰箱雄踞2020年国内万元机市场畅销榜首。而在容声will冰箱的推动下，养鲜已然成为了推动冰箱行业不断前行的驱动力。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;</p><p>2020年，在海鲜市场、冷链运输等环节被新冠病毒屡屡攻破的情况下，容声推出行业首款冷冻冷藏双区除菌净味的双净冰箱，填补了行业冷冻室除菌杀毒的空白。同时，双净系列冰箱也成为了行业的明星系列。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;</p><p>这两大系列的冰箱，根据不同的消费群体和消费习惯形成合力，再加上这两年来，容声冰箱产品力不断提升和优化，已远超行业水平和用户的需求，成功的找到了打开未来市场的核心竞争力，掌握着本轮冰箱升级换新市场的关键筹码。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;</p><p>另外，可以看到海信集团这两年来，在容声品牌塑造和宣传上的不断加码投入，像世界杯体育营销活动带来的全面拉动，足以让容声品牌更具活力，无论是在品牌高端化、年轻化、智能化上，还是在市场占有率方面，都有明显提升。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;</p><p>此次容声新品上市已成为冰箱行业这两年发展的重大亮点，随着消费者冰箱升级需求的升温，容声必将引领冰箱行业的最新需求趋势。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;</p><p>中国家用电器协会秘书长王雷表示，冰箱企业需要思考的关键问题是怎么通过调整产品结构、转型等，给消费者带去他们真正需要的、也真正有价值的产品。容声冰箱对年轻用户的需求深度洞察和悉心满足，再次把他们自身推到冰箱行业技术发展的无人区。其实，小空间和大冰箱，中间差的不是一只羊，而是用户对美好生活的渴求。他们期盼更多的容声们，怀揣初心，与他们同行。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;</p><p class="em_media">（文章来源：青岛财经网）</p>
