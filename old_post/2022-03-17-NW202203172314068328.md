<p>微软智能云矩阵在中国市场的整体服务能力再次翻倍扩容。 </p><p>微软3月17日宣布正式启用微软智能云 Microsoft Azure 新增数据中心区域，这也是由世纪互联在中国运营的第五个微软智能云 Azure 数据中心区域。</p><p>作为中国市场上首个合法合规商业运营的国际公有云，由世纪互联运营的微软智能云 Microsoft Azure 于2012年首次发布，第一批两个 Azure 数据中心区域于2014年3月正式商用。由世纪互联运营的 Microsoft 365现代工作平台、Dynamics 365智能业务应用平台及 Power Platform 低代码开发平台相继于2014年5月、2019年5月、2020年6月在华商用。</p><p>官网显示，目前微软智能云拥有超200个实体数据中心，可部署至全球44个区域，为全球超过10亿客户和两千万家公司提供服务。目前，微软智能云Azure上拥有包括虚拟机、云服务、存储等68个云产品。</p><p>在接受包括21世纪经济报道记者采访时，微软大中华区Azure产品市场部总经理陶然指出，新增 Azure 数据中心区域启用，让微软智能云矩阵在中国市场的整体服务能力再次翻倍扩容。</p><p>“翻倍扩容意味着微软对于中国区云计算服务，以及企业数字化转型的市场需求和客户需求的一个长期乐观的预计。新的区域能够满足未来中国区客户数字化转型、市场转型对云计算的所有需求，有长期充足的市场容量来保证和服务微软的投资。”陶然表示。</p><p>微软大中华区副总裁兼市场营销及运营总经理康容向记者直言，微软看好在中国市场的机会。“首先，越来越多的中国企业想要出海发展，他们想要选择的国际一流的、最可靠、最安全的服务跟产品。另外，当跨国企业想要在华发展，他们也需要有合规、合法的方案来进行。”</p><p>微软最新财报显示，微软第二财季营收为517.28亿美元，与去年同期的430.76亿美元相比增长20%；净利润为187.65亿美元，与去年同期的净利润154.63亿美元相比增长21%。</p><p>其中，微软的智能云部门(包括Azure公共云、GitHub和Windows server等服务器产品)创造了183.3亿美元的收入，增幅为25.5%。作为微软云基础设施服务的Azure，当季收入增长超过46%，成为微软整体的业务中表现颇佳的业务类别。</p><p class="em_media">（文章来源：21世纪经济报道）</p>
