 <p>11 月，特斯拉上海超级工厂启动了特斯拉 Model Y Performance 的交付，中国制造 Model Y Performance 配备了 AMD Ryzen 图形芯片。</p>
 <p>除了 Ryzen 芯片，国产 Model Y Performance 车主还透露，他们的车辆配备了与 Model S Plaid 一样的 12V 锂离子电池。此前，这两款车都采用了铅酸电池，很容易出现故障。</p>
 <p>Teslascope 在 Twitter 上发布的照片显示，最新交付的 Model 3 和 Model Y 也配备了 AMD Ryzen 芯片，新车的 12V 电池也明确标注为锂离子电池。新的 Model 3 和 Model Y 车辆还设有一个小凹槽，用于放置前置摄像头。</p>
 <p>社交媒体上分享的照片是 2021 年 12 月 21 日制造的汽车，暗示特斯拉最近开始为 Model 3 和 Model Y 配备最新的信息娱乐系统。升级后的车辆的 VIN(车辆识别号码)以 139xxx 开头。一些 VIN 码为 137 开头的车主称，他们的汽车没有 AMD Ryzen 芯片或其他升级。</p>
 <p>IT之家了解到，据数码博主 @硬哥分析，相比于之前的英特尔处理器，AMD Ryzen 的 CPU 性能会高出 4 倍、GPU 的性能高出约 8 倍；而和高通最新款车机处理器 8155 相比，其 CPU 性能大约是其 2 倍，GPU 性能大约是 1.5 倍，能为车主带来更流畅的车机使用体验。</p><p class="em_media">（文章来源：IT之家）</p>
