 <p>IT之家了解到，磷酸铁锂电池已经在中国制造的特斯拉车辆上使用，美国市场的 Model 3 也即将采用，如果车主想尽可能长时间保持电池的健康，特斯拉有几个重要的建议。</p>
 <p>修订后的美国 Model 3 车主手册称，“如果你的车辆配备了磷酸铁锂电池，特斯拉建议你将充电限制设置为 100%，即使是日常使用，你也要每周完全充电到 100% 至少一次。”</p>
 <p>这与配备三元锂电池组的 Model 3 电动车车主的习惯不同，特斯拉建议配备三元锂电池组的的车辆大部分时间充电到 80% 或 90%，如果长途旅行的话才建议冲到 100%。</p>
 <p>另一项建议涉及 Model 3 停放超过一周的情况，在这种情况下，特斯拉建议像平时一样驾驶，并在方便的时候尽早充电到 100%。</p>
 <p>有趣的是，特斯拉还建议车主不要在安全地点启动哨兵模式，“此外，一个最佳做法是，在可能的情况下禁用哨兵模式，让 Model 3 定期‘睡眠’。可以使用排除家庭、排除工作地点设置，以防止哨兵模式在不需要的地方自动启动(见哨兵模式)。”</p>
 <p>根据特斯拉的说法，遵循上述指导可以最大限度地提高续航里程，并提高车辆准确确定充电状态和预估续航里程的能力。</p><p class="em_media">（文章来源：IT之家）</p>
