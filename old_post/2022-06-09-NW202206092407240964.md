<p>本期投资提示：</p> <p>AHP 得分——龙芯中科2.78 分，总分的44.1%分位。考虑流动性溢价因素后，我们测算龙芯中科AHP 得分为2.78 分，位于科创体系AHP 模型总分的44.1%分位，位于中游偏上水平。假设以65%入围率计，中性预期情形下，龙芯中科网下A、B、C 三类配售对象的配售比例分别是：0.0474%、0.0419%、0.0231%。</p> <p>实现指令系统自主创新，构建独立的产业生态。龙芯中科是国内唯一坚持基于自主指令系统构建独立于 Wintel 体系和 AA 体系的开放性信息技术体系和产业生态的 CPU企业。在芯片研发方面，公司已独立研制出包括系列化 CPU IP 核、GPU IP 核、内存控制器及 PHY、高速总线控制器及 PHY 等上百种 IP 核；另外，龙芯中科于2020 年推出了自主指令系统 LoongArch(龙芯架构)，基于LoongArch 指令系统的芯片产品龙芯 3A5000 已于 2020 年年底完成流片，并于2021 年 5 月形成销售，产品能性逼近开放市场主流产品水平；在基础软件方面，公司基于 LoongArch 自主指令系统，形成了面向信息化应用的基础版操作系统 Loongnix 和面向工控类应用的基础版操作系统 LoongOS，是全世界范围内极少数建立形成了完整的基础软件技术生态体系的CPU 公司。在芯片生产方面，公司通过设计优化提升性能，摆脱对最先进工艺的依赖。</p> <p>通过自主设计 IP 核，克服境内工艺 IP 核不足的短板。</p> <p>工控类芯片及信息化类芯片双线发力，应用场景不断丰富。公司的“龙芯”系列是我国最早研制的通用处理器系列之一，于2001 年在中科院计算所开始研发，得到了中科院、国家自然科学基金、863、973、核高基等项目的大力支持。在工控系统自主化市场和信息系统自主化市场，龙芯中科均处于市场前列。在工控系统领域，主要应用于网安通信、能源、交通等行业领域，国内已有上百家主要工控和网络安全设备厂商推出了基于龙芯CPU 的工控和网安产品；在信息系统领域，主要面向电子政务、金融、电信、教育等行业，基于龙芯的计算机与服务器已经在各类办公业务系统中应用达百万套以上。</p> <p>目前龙芯CPU 在关键信息基础设施领域应用还处于试点和应用验证的初步阶段，但其通用处理器性能已经逼近商用领域市场主流产品水平，操作系统已经趋于成熟稳定，将在进一步迭代优化后，走向商用领域开放市场。另外公司也正在研发电源芯片、时钟芯片等，以上产品也有望支撑公司利润增长。</p> <p>CPU 行业国产替代空间广阔，龙芯中科有望充分受益。近两年采用国产 CPU 的桌面和服务器产品发展迅速，2020 年中国服务器出货量为 350 万台，同比增长 9.80%，但市场份额仍不足 5%，增长空间巨大。公司通过长期积累，自主研发了包括处理器核心在内的上百种核心模块，取得了 400 余项专利，技术优势突出，产品竞争力较强，处于国内通用处理器行业的领先地位，推出了自主指令系统，掌握了 CPU IP 核的所有源代码，形成了面向信息化和面向工控类应用的基础版操作系统，助力客户摆脱 Wintel和 AA 生态体系的历史包袱，有望加速CPU 国产化进程，享受国产替代带来的红利。</p> <p>业绩增速低于可比均值，研发投入占比逐年提升。2019-2021 年公司营收和归母净利体量较小，且增速低于可比均值。毛利率位于可比公司中游水平，研发投入占比逐年提升。</p> <p>风险提示：龙芯中科需警惕客户及供应商集中度过高、转向自主指令系统后市场开拓不及预期、毛利率波动、政府补助变化、应收账款余额增加导致的坏账增加等风险。</p><p class="em_media">（文章来源：上海申银万国证券研究所）</p>
