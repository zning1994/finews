<p>由于中概股连续大跌，近期一些重仓中概股的机构近期也处于风口浪尖。作为国内较早出海、并在海外市场多有布局的景林资产也特别受到关注，近日就有一则来自社交平台的信息，曝光了一些投资人与景林沟通会的细节。</p><center><img src="https://dfscdn.dfcfw.com/download/D25001498756101415033_w1107h732.jpg" alt="image" style="border:#d1d1d1 1px solid;padding:3px;margin:5px 0;" /></center><p>根据私募排排网的数据显示，2月11日，景林资产在美国证券交易委员会(SEC)公布了去年四季度末的美股持仓情况。统计数据显示，截至去年底，景林资产在美股市场持有34家公司的股票，持股总市值为27亿美元，其中中概股占据相当比重。上述背景下，在近期中概股的调整中，景林资产旗下产品或出现幅度不小的回撤。</p><p>不过，此前景林资产董事总经理高云程就在微信朋友圈发文：“如果你真觉得买的股份是企业的一部分，如果你真觉得这些企业的一部分是你拥有的生意和资产，那就关注这些生意本身的变化。Buy to own the company！ If we can think this way， we will calm down”。</p><p><strong>景林资产持有人沟通会开成了吐槽大会</strong></p><p>由于中概股连续大跌，一些重仓中概股的机构近期处于市场的风口浪尖，备受关注。近期的持有人沟通会上，出现了投资人情绪爆发的局面，有投资人称，“不要把价投当作业绩不好的遮羞布，也不要天天把长远持在嘴上，把行动做出来。”</p><center><img src="https://dfscdn.dfcfw.com/download/D25384497842686150079_w379h712.jpg" alt="image" style="border:#d1d1d1 1px solid;padding:3px;margin:5px 0;" /></center><center><img src="https://dfscdn.dfcfw.com/download/D24796503443634570328_w378h720.jpg" alt="image" style="border:#d1d1d1 1px solid;padding:3px;margin:5px 0;" /></center><center><img src="https://dfscdn.dfcfw.com/download/D24727841744836643001_w383h719.jpg" alt="image" style="border:#d1d1d1 1px solid;padding:3px;margin:5px 0;" /></center><center><img src="https://dfscdn.dfcfw.com/download/D25308120185056081999_w378h720.jpg" alt="image" style="border:#d1d1d1 1px solid;padding:3px;margin:5px 0;" /></center><p>该投资人还提出明确要求，即降规模或者提升基金经理数量。</p><p>还有投资人称，客户持有几年都没有收益，景林公司该暂停固定管理费。</p><p>更有投资人直接怒怼，拜托这个市场总监别在这里扯XX了，先把业绩做上去再说吧。</p><p>“持有几年都没有收益”、“三年不如银行收益高”，一场持有人沟通会俨然成为了持有人的吐槽大会。</p><p>作为国内头部私募机构，景林资产除了在国内运行的阳光私募基金外，其也是较早出海的头部私募，公司在海外市场多有布局。</p><p>公开资料显示，上海景林资产管理公司成立于2004年，是一家以投资境内、外上市公司股票为主的资产管理公司，是在中国证券投资基金业协会登记注册的私募基金管理公司。10余年来，“景林”品牌几乎摘下了国内私募行业以及国外私募基金领域所有重量级奖项。</p><p>在业界，景林资产作为长期价值投资践行者，坚持以自下而上为主的投资策略，以实业投资的心态考量公司的长期价值，并进行全球比较，注重选择高安全边际低估值的公司。这种策略在国内外市场都有长期优秀的历史投资业绩。投资上，景林资产的投资领域主要集中在消费品及现代服务、医疗医药和健康、TMT、先进制造四个行业，投资范围涵盖A股、港股、美股中概等。操作上，基本不择时，长期保持高仓位运作，持股周期长。</p><p>尽管景林极其低调，但其长期稳定的“高调业绩”持续吸引者投资人。 可是如今天怎么就翻船了呢？</p><p><strong>景林资产栽在了哪些股？</strong></p><p>根据此前景林资产向美国证券交易委员会(SEC)提交的去年四季度末美股持仓情况，私募排排网的数据显示，截至2021年四季度末，景林资产美股持仓市值为26.96亿美元。从持仓变动来看，当期景林资产在美股抄底了中概互联网企业、新能源个股，与此同时，清仓了教育股。</p><p>详细来看，景林资产前十大重仓股的市值达到21.23亿美元，在整个持仓中市值占比78.63%。在前十大重仓股中，中概股就有6只，分别是网易、贝壳、中通快递、拼多多、京东、BOSS直聘，持股市值为9.79亿元。</p><p>逆势加仓互联网个股，是景林资产去年四季度的动作之一。在其前十大重仓股中，景林资产增持了70万股拼多多至271万股，增持51万股京东至126.54万股，增持74万股BOSS直聘至206.53万股。DOORDASH与BOSS直聘也新晋前十大重仓股名单。</p><p>但受美国《外国公司问责法》等因素冲击，近期中概股一度集体大幅回调。仅3月以来，中概股累计蒸发就一度超过2万亿人民币的市值。追踪中概股的纳斯达克中国金龙指数更下跌超10%，创2008年10月以来最大跌幅。</p><p>具体来看，截至3月14日，在景林持仓前十的6只中概股中，自年初以来，网易跌近30%、贝壳跌超50%、中通快递跌超25%、拼多多跌超55%、京东跌近40%、BOSS直聘跌近55%。</p><center><img src="https://dfscdn.dfcfw.com/download/D25244609677512073706_w865h479.jpg" alt="image" style="border:#d1d1d1 1px solid;padding:3px;margin:5px 0;" /></center><p>上述背景下，成立于2006年10月、运行时间超过15年的景林资产代表基金“深国投-景林稳健证券投资”，截至2月28日近一年整体回撤就接近20%。</p><p>不过，鼎钜股权投资基金执行董事肖全永就认为，景林属于大基金，在投资协议中，一般会签署3-5年的投资封闭期。</p><p>“这相当于给了基金管理人一个安全垫。即便投资者知道外围市场大跌，也不可能赎回，因为有操作封闭期。但一些小私募基金，资金有优先级和劣后级，以及止损线的要求。一旦触发止损线，就会斩仓，形成多杀多的局面。”肖全永表示道。</p><p><strong>中概股短期有望“回血”</strong></p><p>年初以来，市场大幅震荡，呈持续下跌之势，但与此同时，公募、私募也纷纷自购其旗下产品“稳定军心”，景林资产也是其中一员。</p><p>在1月28日，景林资产发布公告称，基于股市对中国经济基本面和资本市场长期健康、稳定发展的坚定信心，上海景林资产管理有限公司及相关基金经理，将于本公告发布之日起30个交易日内申购公司旗下基金产品，申购金额合计不少于1亿元，持有时间不少于3年。</p><p>在近期市场的调整中，景林资产董事总经理高云程也在微信朋友圈发文：“如果你真觉得买的股份是企业的一部分，如果你真觉得这些企业的一部分是你拥有的生意和资产，那就关注这些生意本身的变化。Buy to own the company！ If we can think this way， we will calm down”。</p><p>中信建投陈果表示，近期中概股的大幅下跌，主要是因为中概股主营业务在国内、但在美国上市的特殊性，股价易受国际形势的影响。2022年3月10日中概股的集体深度回调为情绪面因素推动，主要因为美国“预退市名单”的公布，挫伤了部分投资者对于中概股未来走势的信心。</p><p>中信建投认为，中概股整体上基本面并未出现较大问题，相关财务披露问题并非新的事件，此次回调更多是受到地缘冲突影响引发情绪波动。中概股上市公司的基本面等状况并未发生大的变化等因素，因此短期存在反弹空间，还需进一步跟踪国际形势的变化如中美监管层就财务披露细则等沟通和合作进展。</p><p>瑞银在最新发表的中国市场策略报告中也表示，市场抛售已过度。该行指出，机构投资者似乎是这次中概股主要卖家。对于已作双重上市的中概股，近三个月机构投资者对相关中概股的ADR所有权下降了2.6个百分点，但同时，机构投资者在香港上市部分持股增加0.9个百分点，表明部分投资者已将其ADR转换为H股。从港交所中央结算系统数据也显示，双重上市公司的香港上市部分已从流通股的48%增加到53%，而转换大多发生在京东、阿里巴巴和理想汽车。</p><p>展望后市，市场主流机构表示，本次金融委专题会议均给出了明确部署，投资者情绪得到较大改善，悲观情绪得到释放，短期反弹可期，A股已经进入寻底、筑底过程；多家基金公司表示“最恐慌的时候正在过去，未来一个月市场迎来阶段性修复窗口。”</p><p
