 <p>近日，微软现象级产品“IE浏览器”官宣即将退役，其27年的生涯终将画上句号。</p>
 <p>21世纪经济报道记者梳理发现，早在2015年3月，微软便已确认将放弃IE品牌。今年3月，微软在Windows消息中心再次发出新的警告，提醒用户IE浏览器将于6月正式退役；两个月后，IE浏览器6月16日退役的消息由“微软Edge浏览器”再次官宣。</p>
 <p>从微软不再支持IE浏览器的动作数起，IE浏览器耗时7年才彻底退出历史舞台。5月18日，微软方面向21世纪经济报道独家回应表示，开发一款能够满足当前和未来需求的浏览器需要时间，“在2022年6月15日正式停用前，微软做了一系列准备工作，包括发布基于Chromium的Microsoft Edge、让消费者做好过渡到新浏览器的准备。”</p>
 <p>艾媒咨询CEO张毅向21世纪经济报道记者表示，IE浏览器的退役是为Edge浏览器让路，更是微软在商业模式创新、盈利载体升级换代的直接体现。</p>
 <p>新旧更替</p>
 <p>一个事实是，IE浏览器的颓势已经持续多年。</p>
 <p>根据美国网站通讯流量监测机构statcounter的数据，IE浏览器的市场份额从2009年1月的64.97%一路跌落，于2012年5月彻底失去浏览器市场的头把交椅——IE的市场份额跌至28.87%，此时Chrome浏览器的份额已达29.15%。自2015年8月起，IE的市场份额保持在10%以下，2022年4月仅为0.39%。</p>
 <p>在艾媒咨询CEO张毅看来，IE败在商业化上。“在IE年代，我们几乎看不到什么广告，它只是一个伴随着操作系统带来的一个自带功能，一不小心变成了一个非常强大的、市占率非常高的产品。要改变IE本身的商业逻辑，(我)觉得很难去承载，因为消费者已经习惯了它干干净净的形象。”张毅认为，商业化对IE浏览器来说并不难，但用户已经形成路径依赖，IE也做过商业化尝试，但始终无法突破。</p>
 <p>因此，在2015年IE市场份额降至10%以下的微妙节点，微软选择用Edge浏览器来承载其商业功能。</p>
 <p>2015年3月，微软表示在新一代Windows操作系统自带的浏览器，将启用新品牌Edge，同年正式宣布放弃IE品牌。但IE浏览器并没有马上退出，而是开启了一场长达七年的告别：2020年，微软宣布将切断IE对部分服务的访问，自8月17日起，IE不再支持微软的在线服务，包括Office365、OneDrive、Outlook等。2021年5月19日，微软宣布IE11桌面应用程序停用。今年3月，微软再次放出了IE退役的风声。</p>
 <p>对此，微软方面向21世纪经济报道记者回应：“我们已经看到，随着新的Web标准和创新的发展，开发浏览器的时候需要全盘考虑，而其在技术上已经超越了Internet Explorer.开发一款能够满足当前和未来需求的浏览器需要时间——这个浏览器就是Microsoft Edge.”</p>
 <p>在张毅看来，七年的缓冲期，主要还是因为此前Edge浏览器没有准备好。与IE相比，Edge的商业色彩可谓是无孔不入，如今经过时间验证，Edge的商业模式并未引发用户反感并大规模启用，因此微软选择此时停止IE浏览器。</p>
 <p>Edge这七年</p>
 <p>七年间，微软致力于打造一款比IE“更快速、更安全、更现代”的新浏览器。</p>
 <p>为了拥抱新的Web标准，Edge选择拥抱新的开源技术。据微软方面向21世纪经济报道记者介绍，微软于 2020 年 1 月 15 日发布了基于 Chromium 内核的 Microsoft Edge，以提供速度、性能、网站和扩展的一流兼容性以及内置的隐私和安全功能，取代了此前改造于IE浏览器Trident内核的EdgeHTML.因此，Edge还提供Chrome的兼容性，具有比IE更高级别的安全功能。</p>
 <p>性能方面，微软方面透露，在Cerner Corporation进行的试验中，与IE相比，Edge将渲染时间缩短了85%，CPU利用率减少了33%，内存利用率减少了32%。此外，Edge还引入了集锦、垂直标签、密码监视器、跟踪防护以及内置购物等功能。</p>
 <p>值得注意的是，微软还为IE与Edge的过渡做了衔接。2019年，微软公布新版Edge浏览器将上线的新功能，其中包括“IE模式”——Edge浏览器直接以标签页的形式整合了 Internet Explorer，这将允许企业在现代浏览器中运行基于 Internet Explorer 的已有应用。</p>
 <p>此外，IE浏览器停用的消息于2021年5月公布后，微软还对Edge的IE模式做出改进，恢复了IE COM对象的调用和实现双向Cookie共享。IE模式作为Edge浏览器与IE浏览器的桥接点，是微软迁移IE用户的一大举措。</p>
 <p>“Microsoft Edge 的 Internet Explorer (IE)模式是唯一一款内置支持基于 IE 的 Web 应用和站点的浏览器，包括对 ActiveX 控件等功能的向下兼容性支持。”微软方面对21世纪经济报道记者称，这意味着，企业机构可以为其用户设置 IE 模式，个人用户也可以根据需要访问 IE 模式。</p>
 <p>目标成为首选浏览器</p>
 <p>5月16日在微博上官宣IE退役当天，微软公布了Edge浏览器的成绩：Edge浏览器桌面市场份额已于近日突破10%。</p>
 <p>微软方面表示，Edge浏览器的目标是成为企业首选的浏览器。“我们在整个浏览器开发过程中的目标是提高客户使用的兼容性，减少 Web 开发人员的碎片化，同时提供新的差异化体验。对于企业和组织机构，我们的目标是成为企业首选的浏览器，在安全性、可管理性和生产力方面带来Windows和Microsoft 365的卓越功能。”</p>
 <p>张毅预测，浏览器很有可能支撑起微软的第三代主营收入。“第一代是卖系统，第二代是卖云计算，第三代可能就是通过浏览器卖广告卖流量入口。”张毅分析，来自其他操作系统的冲击可能减少微软对操作系统收费的期望。在此基础上，Edge可以承担商业功能，而IE已经没有办法承接。</p>
 <p>“Edge相对于IE更流畅、外观更好，且主要一点，有很多扩展功能，能够与Chrome和Firefox竞争。”浙江大学国际联合商学院数字经济与金融创新研究中心联席主任、研究员盘和林表示，微软将在Windows11中提高系统可视化界面的用户视觉观感，其系统也在逐渐向苹果IOS看齐，增强Windows在多设备终端中的通用性。“所以微软是要打开多终端系统应用，而不是局限于PC.”</p>
 <p>不过，尽管微软已经进行一定的多终端布局，张毅认为此举可拓展性不强。“以智能手机为例，在这个终端里，浏览器的角色现在已经弱化”，张毅认为，而这并非Edge浏览器主力拓展的方向。</p><p class="em_media">（文章来源：21世纪经济报道）</p>
