&nbsp; &nbsp; &nbsp; &nbsp; <br /></p><p>面对外贸集装箱空箱和舱位资源持续紧张的趋势，宁波港积极应对疫情、台风等影响，建专班、战疫情、抢生产、保畅通，以“硬核”力量助力全球物流链、供应链畅通。最新数据显示，公司全年完成货物吞吐量9.45亿吨，同比增长2.9%；集装箱吞吐量完成3430.3万标箱，同比增长8.1%。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </p><p>在保障港口生产工作有序进行的同时，宁波港通过开展降本增效、资本运作、投资建设、科技创新等工作全面提质增效，为实现长期可持续发展打下坚实基础。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </p><p>记者从公司获悉，2021年，宁波港积极开展降本增效专项工作，持续优化企业筹资渠道，并在成本管控方面取得显著成效；同时，宁波港继续加速推进资本运作，子公司宁波远洋分拆上市和向招商港口非公开发行股票两个项目先后获得中国证监会受理。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </p><p>投资建设方面，公告显示，宁波港的淡水河谷西三区、穿山港区1号泊位等合资合作项目得到有序推进；穿山1号泊位、通用泊位改造工程等项目完成竣工验收；中宅二期、梅山6号-10号集装箱码头二阶段、梅山滚装配套工程等项目完成交工验收。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;</p><p>科技创新方面，公告称，宁波港梅东公司8号泊位已具备完整作业链自动化运行能力，甬舟公司实现智能集卡常态化编组实船作业，鼠浪湖公司卸船机自动化已进行实船作业测试；同时，宁波港进一步完善绿色港口整体规划，编制《绿色强港指标体系》，并积极推进绿色能源应用、绿色装备升级和节能工艺改造，龙门吊油改电与LNG等实现规模化应用，完成7套高压岸电建设，集装箱和干散货专业泊位岸电覆盖率达85%。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </p><p>公司表示，宁波港2021年持续坚持以码头运营为核心，以港口物流和资本经营发展为重点的经营模式，实现港口生产全面突破，助力全球产业链复苏，并通过降本增效、资本运作、投资建设、科技创新等多项措施进一步巩固其一流强港“硬核”实力，为“十四五”期间的良好发展打下坚实基础。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </p><p>公司强调，面对未来我国港口吞吐量高基数、中低速增长的特点，宁波港将围绕全球重要港航物流枢纽、全球重要大宗商品储运基地、全球重要的海事特色航运服务中心等目标定位，打造国内大循环战略支点、国内国际双循环战略枢纽。加快建设“四个一流”港口，统筹推进“三型五化”发展，进一步增强自身竞争力、服务力和影响力，力争到2025年基本建成世界一流强港新标杆、世界一流港口运营商。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;</p><p class="em_media">（文章来源：中国证券报·中证网）</p>
