<p>1月10日晚，荣耀首款折叠屏手机Magic V正式发布。Magic V搭载高通4nm工艺制程全新一代骁龙8移动平台及荣耀最新发布的操作系统Magic UI 6.0,9999元起售，1月18日10点08分正式开售。</p>
 <center><img src="https://dfscdn.dfcfw.com/download/D24850401980809025687_w1392h610.jpg" style="border:#d1d1d1 1px solid;padding:3px;margin:5px 0;" /></center><center><img src="https://dfscdn.dfcfw.com/download/D25752807180169646448_w2829h1279.jpg" style="border:#d1d1d1 1px solid;padding:3px;margin:5px 0;" /></center><p>据荣耀CEO赵明介绍，荣耀MagicV以荣耀自研的折叠屏解决方案，为折叠屏手机带来折叠形态、旗舰性能、智慧交互三大维度的创新升级。并首发搭载全新Magic UI 6.0带来全场景的智慧操作体验，让折叠机不再是科技尝鲜的玩具，而是“一部到位”的主力机。</p>
 <p>形态方面，荣耀Magic V采用左右内折的内外屏设计，合起来是一部常规比例的直屏手机，外屏搭载6.45 英寸OLED屏，可以让消费者获得类似主流直板旗舰手机的操作体验；展开后为一部高能平板，手机配备的7.9英寸折叠大屏不仅带来更丰富的内容展示，还能通过多应用分屏、悬浮窗口，实现便捷的多任务操作，如一边聊天一边视频通话或看公众号文章等，提升用户使用效率。</p>
 <p>目前，荣耀Magic V能够支持100多款单应用平行视界、1000多款多应用分屏适配；应用市场里排名前100的应用已100%实现大屏适配、排名前1000的应用中已有90%实现了大屏适配。</p>
 <p>性能方面，荣耀Magic V是当前业界唯一搭载高通4nm工艺制程全新一代骁龙8移动平台的折叠屏手机。为了全面发挥全新一代骁龙8移动平台的强大性能，荣耀Magic V还搭载OS Turbo X技术，通过超低时延引擎、抗老化引擎和智慧内存引擎，实现了平台级重构和系统级调校，大幅提升系统流畅性、系统抗老化性和系统功耗表现。</p>
 <p>荣耀Magic V同时还克服了折叠屏手机空间布局难度高的特点，在产品中巧妙的容纳了4750mAh大容量电池，并同时支持66W有线超级快充，实现了更大的续航，更快的充电。“我已经使用了一段时间，Magic V的电池能够保持一天的正常使用。”赵明说。</p>
 <p>折痕和厚度是折叠屏手机的两大痛点，荣耀Magic V采用荣耀自研的超薄悬浮水滴铰链设计，消除了折叠产生的超大缝隙，大幅度降低了厚度，保证了屏幕在多次开合后的长寿命以及轻折痕。荣耀Magic V折叠形态仅有14.3mm厚度、展开形态两侧均为6.7mm厚度，是业界目前最纤薄的横向折叠屏手机之一，航天级用材则保证了铰链开合寿命可达20万次。</p>
 <p>影像方面，荣耀Magic V配备了5000万多主摄融合计算摄影及4200万内外屏双前置摄像头，让荣耀Magic V在折叠及打开状态下，都能够获得优质的自拍及视频效果。安全及隐私保护方面，荣耀Magic V是荣耀首款搭载独立安全存储芯片的手机，还设计了独立于安卓操作系统之外的双TEE安全系统，实现了手机硬件本身和信息数据安全的双保险。此外，荣耀Magic V还采用最新的曲面纳米微晶玻璃工艺，在保证纤薄机身的前提下，大大增加了屏幕的抗摔性。</p>
 <p>荣耀MagicV还首次搭载Magic UI 6.0，赵明介绍，荣耀Magic UI 6.0通过Magic Live智慧引擎将带来多设备之间的智能协同。基于全新HONOR Connect强大的互联能力，不仅支持手机、平板和PC之间的无缝互传，还支持荣耀生态下PC与PC之间的高速互传。</p>
 <p>荣耀MagicV拥有钛空银、燃橙色以及亮黑色三种配色。12+256GB版本售价9999元，12+512GB版本售价10999元。1月10日22:00正式开启预售，1月18日10:08正式开售。</p><p class="em_media">（文章来源：上海证券报·中国证券网）</p>
