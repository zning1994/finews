<p>拥有379家上市公司的科创板，首富悄然易主。 &nbsp; &nbsp; &nbsp; </p><p>2021年12月22日，“光伏小巨人”天合光能(688599)报收75.15元，跌2.02%，总市值1554.12亿元。 &nbsp; &nbsp; </p><center><img src="https://dfscdn.dfcfw.com/download/D25011916780390032191.jpg" alt="天合QQ图片20211223104345.png" title="f4584c1dad0ebf55aaabed8968a38bdb.png" style="border:#d1d1d1 1px solid;padding:3px;margin:5px 0;" /></center><p>“玻尿酸第一股”华熙生物(688363)报收162.57元，涨0.86%，总市值780.34亿元。</p><p>上游新闻记者独家注意到，天合光能实际控制人高纪凡持股市值达到546.86亿元，已经超过了华熙生物赵燕女士的460.87亿元，成为科创板首富。 &nbsp; </p><p><strong>今年前三季净利润11亿多元 </strong></p><p>天眼查数据显示，天合光能注册地为江苏省常州市新北区天合光伏产业园天合路2号，现注册资本20.6803亿元。主营业务为光伏产品、光伏系统、智慧能源三大板块。光伏产品包括单、多晶的硅基光伏组件的研发、生产和销售；光伏系统包括电站业务及系统产品业务；智慧能源包括光伏发电及运维服务、智能微网及多能系统的开发和销售以及能源云平台运营等业务。 &nbsp; &nbsp; &nbsp; </p><p>上游新闻记者获悉，其前身为“常州天合铝板幕墙制造有限公司”，1997年12月26日成立，2002年4月8日，变更为“常州天合光能有限公司”， 2017年11月22日，变更为“天合光能有限公司”， 2017年12月28日，改制为“天合光能股份有限公司”。 &nbsp; &nbsp; &nbsp; </p><p>天合光能2020年6月10日在科创板上市，首发31020万股，发行价为8.16元/股，募资25.31亿元，用于光伏发电项目投资、组件产能扩张及研发中心建设。 &nbsp; &nbsp; &nbsp;</p><p>今年前三季，该公司营收为312.65亿元，同比增长56.9%；净利为11.81亿元，同比增长39.18%；归属于母公司股东的净利为11.56亿元，同比增长39.05%，这是天合光能业绩新高。 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </p><p><strong>高纪凡1997年创业曾在美上市 &nbsp; </strong></p><p>上游新闻记者获悉，高纪凡，男，1965年生，中国国籍，硕士研究生学历。 &nbsp; &nbsp; &nbsp; </p><p>高纪凡1985年毕业于南京大学化学系高分子专业，获化学学士学位，1988年，毕业于吉林大学量子化学专业，师从量子化学家唐敖庆教授，获物理化学专业硕士学位。</p><p>1988年9月至1989年3月，高纪凡就职于常州东怡联合开发公司；1989年3月至1989年9月，就职于常州对外经济技术贸易公司；1989年10月至1992年5月，担任广东顺德大良清洗剂厂副厂长；1992年6月至1997年10月，任武进协和精细化工厂厂长。</p><p>1997年12月至2017年12月，高纪凡担任常州天合光能有限公司董事长、总经理；2006年至2017年，担任Trina Solar Limited(开曼天合，系美股，已经退市)董事长、首席执行官；2017年12月至今，担任天合光能董事长、总经理。</p><p>高纪凡还担任中国光伏行业协会理事长、全球太阳能理事会联席主席、中国机电产品进出口商会副会长等职务。 &nbsp; &nbsp; &nbsp; </p><center><img src="https://dfscdn.dfcfw.com/download/D24782235659984503791.jpg" title="7ba79d157904a9b420bf8570d5521375.png" style="border:#d1d1d1 1px solid;padding:3px;margin:5px 0;" /></center><p>高纪凡通过直接及间接控制天合光能 40.86%的股权，其中直接持股 17%，间接通过其控制的盘基投资、清海投资和天合星元持股 19.19%，并通过一致行动人常州锐创、常州携创、常州赢创、常州凝创、常州天创、吴春艳、有则科技合计控制 4.67%。 &nbsp; &nbsp; </p><p>这样，高纪凡持股数约占天合光能总股本的35.19%，按照75.15元股价，折合市值约 546.86亿元。 </p><p><strong>股价今年累计上涨227% &nbsp; &nbsp; </strong></p><p>上游新闻记者了解到，天合光能实际控制人高纪凡之所以成为科创板首富，得益于天合光能今年大涨。 &nbsp; &nbsp; &nbsp; </p><p>在光伏行业整体上涨的背景下，业绩优秀的天合光能股价今年累计上涨达227.17%，从去年底的23.15元，涨到如今75.15元。 &nbsp; &nbsp; &nbsp; </p><p>而同期华熙生物也上涨，但涨幅只有11.33%，因此，尽管华熙生物实际控制人赵燕持股比例高达59.06%，但持股市值460.87亿元，难以匹敌高纪凡。 &nbsp; &nbsp; &nbsp; </p><p>目前科创板排名第三的富豪，为格林微(688728)赵立新、曹维夫妇，持股比例为44.36%， 市值334.07亿元。而杭可科技(688006)曹政、曹骥父子排名第四，持股比例为69.39%，市值 291.91亿元 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </p><p><strong>延伸阅读 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </strong></p><p><strong>科创板面世两年已经诞生四位首富</strong></p><p>江山有代才人出，各领风骚三五月。</p><p>上游新闻记者获悉，2019年7月22日，科创板首批25家企业集体亮相上市，当天收盘，华兴源创实际控制人陈文源、张茜夫妻持股市值186亿元，成为科创板首富。 &nbsp; </p><p>据介绍，2005年，37岁的陈文源与妻子一起凑了100万元成立了华兴源创。作为配套企业，华兴源创获得了苹果、三星、LG等明星平板厂商的青睐。</p><p>当年 11月6日，“玻尿酸第一股”华熙生物上市，华熙生物实际控制人、董事长兼总经理时年53岁的赵燕女士，持股市值达到241亿元，超过陈文源、张茜夫妻，成为科创板首富。 &nbsp; &nbsp; &nbsp; &nbsp; </p><p>早在23岁时，赵燕就到海南淘得了人生第一桶金，随后又到北京打造了SK大厦、五棵松体育馆等一批标志性建筑，在地产和投资领域风生水起。</p><p>2001年其花45万元买断玻尿酸配方，靠着这个一瓶成本几十元的产品，华熙生物迅速发展成为全球最大的玻尿酸生产研发企业。</p><p>寒武纪创始人陈天石是第三位科创板首富。 &nbsp; &nbsp; &nbsp; &nbsp; </p><p>陈天石出生于1985年，其自称从小沉迷打游戏，大学时甚至一度因成绩太差抓阄选择考研专业。其实，他与哥哥陈云霁一样，都是学霸级人物。</p><p>陈天石16岁考入中科大少年班，20岁出头博士毕业，31岁创立寒武纪并研发出首款人工智能芯片，35岁推动公司上市，2020年7月20日，寒武纪成为“AI芯片第一股”，仅两天后，公司市值破千亿，陈天石以327亿元身家，成为科创板首富。 &nbsp; &nbsp; &nbsp; &nbsp; </p><p>但上市之后，寒武纪市值逐渐缩水，不久，科创板首富宝座重归“玻尿酸女王”赵燕。</p><p>如今，天合光能高纪凡，成为第四位科创板首富。</p><p class="em_media">（文章来源：上游新闻）</p>
