<p>4月11日，vivo举行春季新品发布会，发布大屏旗舰vivo X Note以及vivo首款折叠屏手机vivo X Fold，vivo X Note售价5999元起，vivo X Fold折叠屏手机售价8999元起。发布会结束后，这两款新品正式在京东开启预售，vivo X Note将于4月15日全面开售。此前加入京东vivo X Note先行者计划的用户，在4月12日23:59前支付尾款即可享受抢先发货权益，尝鲜快人一步。</p><p>同时，为了让用户更好的“尝鲜”新品，针对vivo X Note京东还推出“全能换新”、“全能保修”和“不限次碎屏免费修”服务保障，包括100%保值换新、30天无忧退、365天意外换新、1年不限次免费维修等服务，还有至高24期免费分期，购机晒评论赠无线充，让广大消费者换机更省心。</p><p>vivo X Note作为vivo新一代高端旗舰，拥有最顶的配置性能，屏幕方面配备一块7英寸三星E5材质打造的双曲面屏，拥有21:10黄金宽比例，分辨率达到3080×1440，支持120Hz自适应刷新率，视觉效果非常震撼。核心配置方面，vivo X Note搭载了骁龙8 Gen1旗舰芯片，内置了5000mAh大电池，支持80W双电芯闪充，50W无线闪充，随时随地，电不离身，全程高能，一扫续航焦虑。值得一提的是，vivo X Note屏幕尺寸虽然大，机身却非常轻薄，整机重量216g(素皮)，厚度为8.75mm，而且整机采用重心分移技术，首创一体化拱桥结构设计，当大屏旗舰实现纤薄手感，让你获得视觉与手感的双重享受。配色方面，vivo X Note拥有晴山蓝、大地灰、璨夜黑三款配色可选。</p><p>值得一提的是，vivo X Note还配备了3D大面积超声指纹，指纹识别区域是单点光电指纹的11.1倍，识别区域高达578mm2，基本可以盲解，而且录入过程只需要按压一次，手指沾水也依然能快速识别。影像方面，vivo X Note也同样值得期待，搭载了蔡司四摄模组，分别为5000万像素OIS超大底主摄、4800万像素114°超广角主摄、1200万专业人像镜头、1200万专业人像镜头以及一颗800万像素60倍OIS潜望式超远摄，全系镜头标配蔡司T*镀膜认证。</p><p>vivo X Fold载新一代骁龙8 Gen1 SPU定制旗舰芯片，采用了航天级浮翼式铰链，锆合金浮动中板结构，有效降低手机的凹陷，采用内外双120Hz自适应刷新，额外节省功耗33.3%，触控响应提升20ms，有效缓解卡顿和掉帧。vivo 表示，vivo X Fold获得全球首个30万次TUV莱茵折叠无忧认证，且在三十万次折叠后，依旧能保持良好的使用体验。影像方面，vivo X Fold搭载了全焦段蔡司四摄，配合vivo影像算法矩阵，可以轻松地完成专业影像创作。此外，作为全球首款配备双屏幕指纹的折叠屏，vivo X Fold拥有3D超声双屏幕指纹，识别速度快、准确度高；加上指纹配对算法能力，可实现单屏录入，双屏使用，解锁从此变得简单。</p><p>想要入手vivo新款手机的朋友，即日起上京东预订，享多重安心服务，换新更放心。</p><p class="em_media">（文章来源：青岛财经网）</p>
