<p>折叠屏手机爆发之际，行业硝烟再起。</p><p>昨晚(4月11日)，折叠屏手机领域又迎来了一个新玩家，vivo携首款折叠屏手机vivo X Fold闪亮登场，定位高端商务旗舰，至此，国内主流手机厂商已全部入局。</p><p>vivo是这场战事中最后一个登场的国内主流手机厂商，然而它一出场便宣告要开启折叠屏手机2.0时代。</p><p>过去的一年，vivo刚刚以21.5%的市场份额夺冠中国智能手机市场，但缘何在折叠屏手机市场却慢人一步，成为了“压轴者”？这到底是vivo跟风创新的姗姗来迟，还是一场蓄谋已久的志在必得？初来乍到便要引领行业，vivo又胜算几何？</p><p><strong>vivo X Fold开启折叠屏手机2.0时代 </strong></p><p>去年年底至今，各主流手机厂商纷纷推出自家折叠屏手机新品，并快速将其价格降至万元以内，一度掀起折叠屏手机热潮，一时间，折叠屏手机开始由过去的尝鲜逐渐走向大众视野，与此同时，陡然大增的市场需求，令缺货、加价成为了这一领域的常用标签。</p><p>不过，尽管这一波折叠屏手机新品较几年前华为、三星等率先推出的折叠屏手机在技术成熟度、产品体验等方面已有了巨大进步，价格优势也尤为明显，但折叠屏手机折痕、续航、应用场景等行业痛点依然难掩，行业仍处于创新探索的1.0时代。</p><p>而今，昨日问世的vivo首款折叠屏手机vivo X Fold却宣称将通过一部“更成熟更专业的折叠屏真旗舰”开启折叠屏手机2.0时代。</p><p><strong>vivo X Fold有哪些过人之处？从昨日刚刚举行的vivo X Fold发布会了解到，首先，尽管vivo X Fold为vivo在折叠屏手机领域的开山之作，但它一诞生便有着非常明确的产品定位，为备战高端商务市场而来，有别于市场其他折叠屏产品。</strong></p><p>据介绍，在商务场景中，vivo X Fold展开后的平板级视野为在线浏览、应用多开等使用场景提供极大便利。L型悬停在视频通话、阅读/输入等环境中打造更加便捷的使用体验，配合手机内置的智慧办公套件，在商务环境下为消费者解放双手，高效释放生产力。同时，vivo X Fold可以实现外屏预览帮拍、腰屏显示拍摄、后置高清自拍等摄影新玩法，让消费者享受不一样的影像创作乐趣。</p><p>vivo X Fold还能轻松驾驭股票投资场景，借助8英寸内屏的广阔视野，vivo联合东方财富、大智慧、同花顺、富途牛牛、老虎证券、腾讯自选股等六家用户常用的三方金融股票软件深度调校，可助力用户金融投资场景下的信息获取、交流、操作等行为，让金融人士的使用体验更流畅、更高效。</p><p>其次，vivo X Fold集合了多项行业领先技术，令用户在屏幕体验、铰链技术、创新应用等层面全面提升。</p><p>铰链是折叠屏手机的关键，vivo X Fold采用了航天级浮翼式铰链，在打造过程中，使用了液态金属锆合金、F53航空高强度钢、碳纤维板等六重航天级材料，能使铰链平整度更好，重量轻，抗击外部冲击能力更强，同时运用锆合金浮动中板、UTG玻璃、行业最小左右弯折半径(2.3毫米)打造折痕优化解决方案，使屏幕折痕更浅，最终经莱茵测试，vivo X Fold在经受30万次弯折测试后，折痕依旧控制得很好。</p><center><img src="https://dfscdn.dfcfw.com/download/D25303167068063653078_w1080h608.jpg" alt="Image" width="580" style="border:#d1d1d1 1px solid;padding:3px;margin:5px 0;" /></center><p>消息人士透露，vivo X Fold折叠屏手机所采用的铰链供应商为苹果供应链厂商杭州安费诺飞凤通信，这款铰链仅硬件成本就超过1200元，是目前市面上折叠屏铰链成本之最。</p><p>在屏幕方面，vivo X Fold乃全球首款3D超声双屏幕指纹的折叠屏手机，抗环境干扰能力更强、解锁速度更快、安全性更高。vivo曾是屏幕指纹的开创者，这一技术业已成为行业标配，如今vivo再次引领行业，将双屏幕指纹技术引入折叠屏手机。屏幕大小方面，vivo X Fold搭载8英寸超感折叠巨幕内屏和6.53英寸曲面外屏，拥有平板级内屏和直板手机的外屏体验。</p><center><img src="https://dfscdn.dfcfw.com/download/D25731715923820951206_w1080h607.jpg" alt="Image" width="580" style="border:#d1d1d1 1px solid;padding:3px;margin:5px 0;" /></center><p>性能方面，vivo X Fold采用骁龙8 Gen1 SPU定制旗舰芯片，性能拉满的同时，安全性也进一步得到保障。同时，vivo X Fold配备4600mAh电池和66W双电芯闪充，是行业首款搭载50W无线闪充的折叠手机，同时支持10W反向快充，可解决商务人士的充电焦虑。</p><p>如此专业、成熟的折叠屏旗舰，其价格也令人心动，vivo X Fold12GB+256GB版本8999元，12GB+512GB版本9999元。</p><center><img src="https://dfscdn.dfcfw.com/download/D25094630381476519748_w1080h591.jpg" alt="Image" width="580" style="border:#d1d1d1 1px solid;padding:3px;margin:5px 0;" /></center><p>3月30日，博鳌亚洲论坛与vivo联合宣布，双方成为战略合作伙伴，vivo X Fold成为博鳌亚洲论坛官方指定手机。</p><p><strong>一场蓄谋已久的志在必得 </strong></p><p>vivo X Fold惊艳亮相引发市场广泛关注的同时，外界也普遍产生了一个疑惑，vivo缘何选择此时推出折叠屏手机？毕竟，在此之前，市场上除苹果之外的主流手机厂商，普遍已推出折叠屏手机，vivo在推出速度上已相对落后。</p><p>对于折叠屏手机的“晚到”，在vivo产品副总裁黄韬3月28日发布的微博长文中其实已经有了一些说明。</p><p>黄韬透露，其实早在四年前，vivo便已开始预研折叠屏，期间前前后后推敲过几十版方案，vivo不断地自我否定，因为它们和公司想带给用户的“完美”折叠屏旗舰有不小的距离。</p><p>vivo认为，折叠屏，屏是重中之重，vivo不能因为折叠形态，而牺牲屏幕体验，也不能因为长期频繁开合导致折痕不断加深而影响观感，这其中就涉及屏幕选型、调教、铰链技术等复杂系统工程，另外，vivo作为屏幕指纹手机的开创者，也希望将屏幕指纹引入折叠屏手机，但在折叠屏上实现屏幕指纹并非易事，牵扯到材料学、声学和结构，非常复杂。</p><p>“四年里，客户在催，用户在盼。但我们不能急，一急就容易忘了初心，乱了方寸，我们甚至想过放弃。”黄韬在微博中表示，vivo一直坚信，只要能提供最好的体验，即便晚一点，大家也愿意等，“晚了一点，但好了很多。相信当你们看到vivo X Fold这款产品的时候，能感受到我们的用心和坚持。”</p><p>黄韬说，他很喜欢乔布斯的一句话，大致意思是，你不曾见过他们，不曾彼此聊起彼此的故事，但当他们拿着你倾注了真心与爱，所创造的产品，他们总能心领神会。</p><p><strong>事实上，从市场角度来看，vivo选择此时推出折叠屏手机，乃明智之举。</strong></p><p>第一，经过了前期各手机厂商的市场培育，外界对折叠屏手机已有了一定认知，折叠屏市场在今年迎来了爆发元年，市场需求爆发之际，vivo推出稳定成熟的折叠屏产品vivo X Fold可谓顺应了天时。</p><p>第二，经历了去年年底至今年年初的缺货、加价潮后，折叠屏产业链产能目前已得到大幅提升，当前折叠屏手机行业缺货现象也已大幅缓解，vivo此时推出折叠屏手机，不用担心需求过旺导致的产能跟不上的问题。</p><p>第三，前几个月一些从尝鲜到大众的折叠屏手机产品陆续面世后，折叠屏手机这一产品类型的优缺点、以及用户需求已然显现，vivo针对痛点、缺点推出了更成熟、专业、完善的折叠屏产品，能更好地赢得消费者喜爱。</p><p>第四，vivo X Fold一经推出便精准定位高端商务市场，或有意承接前两年华为、三星折叠屏手机用户的换机潮，以助力vivo更好地发力高端市场。</p><p><strong>低调潜行，慢亦快 </strong></p><p>表面上看，vivo在折叠屏手机领域推出慢，或因为其研发创新、技术落后于同行，但真正了解vivo的人都知道，低调潜行、行稳致远是vivo一贯的作风，因此vivo X Fold的姗姗来迟，走的也正是vivo自己的风格。</p><p>据了解，十多年前，vivo就一直强调要坚持“拥抱用户，追求极致，创造惊喜”，十多年后的现在，vivo依然初心未变。</p><p>今年年初，vivo创始人沈炜发表题为《拥抱用户，追求极致，创造惊喜》的2022年新春致辞。他表示，2021年，对vivo来说，是极具转折意义的一年，2022年，将是智能手机行业最具挑战的一年，但无论顺境还是逆境，vivo都不能忘记初心，要一切回归到以用户为导向，敢于追求极致，持续创造惊喜，做到人无我有，人有我优，在手机这个科技属性极强的行业里，形成“持续主动引领，绝不被动跟随”的vivo style。</p><p>过去十多年，vivo把“敢于追求极致，持续创造惊喜”深入骨髓，始终在手机领域默默耕耘，为行业带来了不少首创惊喜。</p><p>从2012年vivo X1以全球最薄机身、全球首款Hi-Fi音效等亮点惊艳北京水立方全场，再到后来行业首款2K屏、Android首款OIS、首款屏幕指纹、首款前置柔光双摄、首款升降摄像头、首款微云台防抖、蔡司T*光学镜头等，还有诸多引领行业的交互体验创新：访客模式、单手模式、smartwake、微信双开、分屏多任务、超级截屏……vivo对手机行业的贡献和成果不一而足，但它却鲜少张扬。</p><p>2021年，面对中国市场第一的好成绩，vivo亦如是。来自IDC等权威调研公司的数据显示，2021全年vivo以21.5%的市场份额稳居中国市场第一，年出货量超7100万台，同比增长23.3%。与此同时，在印尼、印度、泰国、菲律宾等海外市场，vivo手机出货量均名列前茅。对此，vivo创始人、CEO兼总裁沈炜在年初的新春致辞中，也仅轻描淡写地说“取得了一点成绩”，且外界也未见vivo夺冠中国手机市场的信息频繁见诸报端。</p><p>最近几年，vivo发展显著提速，目前，vivo已覆盖全球4亿多用户，服务60多个国家和地区。2021年取得中国市场第一后，vivo将其定位为质变元年，这意味着后续几年vivo发展的步伐或将更快。</p><p><strong>稳扎稳打的高端突破之路</strong></p><p>折叠屏手机之于手机行业至少有两重意义，第一，代表着手机行业下一个创新突破方向和竞争角逐之地。此前几年，全球各手机厂家普遍进入了一场拼参数的同质化竞争格局中，纷纷在影像、充电、屏幕等方面突围，相关领域已几达天花板，而今折叠屏手机已成为各企业下一个创新突破点。</p><p>第二，折叠屏手机承载着国产手机企业弯道超车高端手机市场的梦想。一直以来，苹果、三星占据全球主要的高端手机市场，而高端是行业利润最丰厚的一端，因此各参与企业均纷纷觊觎。当下苹果并未推出折叠屏手机，国产手机企业有望通过领先推出的折叠屏手机竞得一部分高端手机市场。</p><p>通过打造成熟、专业的折叠屏旗舰手机vivo X Fold，站稳高端市场脚跟，也是vivo高端突破之路上的重要一环。</p><p>过去两年来，vivo为冲击高端市场，从战略、产品、研发、品牌等齐齐发力，稳扎稳打。</p><p>2019年vivo进行战略调整，明确了公司产品优先，研发优先的理念，强调要在用户体验的几个“硬骨头”上下功夫。经过用户动态研究和技术动态研究，vivo把未来需要发力的长赛道进行梳理，总结出四大方向：设计(包括外观、交互等)、影像(拍摄、视频等)、系统(包含底层系统和AI技术使用等)、性能(游戏等场景)。</p><p>vivo所谓的长赛道，有两个特点：一是消费者持续关注，有持续的高要求；二是在这个赛道上，vivo可以持续迭代，有可为的空间。</p><p>围绕这一战略，vivo打造产品规划、技术规划到技术预研这样一套拥有完善流程的产品“铁三角”，不惜一切代价进行研发投入，后续又成立了中央研究院，以及各级研发中心，致力于36个月以上的技术规划，专注新产品形态跟踪研究、国内外行业前沿技术扫描、影像、芯片等领域的技术布局，让vivo始终走在行业技术的前沿。</p><p>为更好地进入高端市场，vivo在影像领域与蔡司合作，将传统光学技术与计算影像研发结合，让蔡司镜头能够被广泛地用于现在这个分享、创作的时代，提供了更全面的影像功能，降低了普通人进入专业影像的门槛；同时为服务用户拍照和录像的需求，vivo还主导开发高速计算成像的专业影像芯片V1，形成了一套vivo自己的影像处理系统，并服务vivo手机影像长赛道；此外，vivo手机还普遍搭载针对手机防抖的解决方案微云台，将手持云台嵌入到手机设备中，有助于提高摄像质量。</p><p>从品牌角度看，vivo目前有两个终端品牌，分别是vivo和主攻互联网极客人群的iQOO，vivo品牌下又分为多个产品系列，分别是定位专业影像旗舰的X系列、时尚自拍旗舰的S系列和入门的Y系列等，品牌定位和产品定位相对清晰稳健，覆盖用户面较广。</p><p>昨日，vivo产品副总裁黄韬在发布会上透露，vivo X Fold的推出，恰逢X系列推出十周年，目前X系列累计用户已超过1亿。</p><p>另据Counterpoint数据显示，2021年底，在中国市场4000-5000元的价位段，vivo以24.1%的市占率位居第一 ，Counterpoint也将评价vivo“在中国中高端领域取得重大进展”。</p><p>“科技公司的高端突破是一场没有捷径的马拉松，只有靠厚积薄发和稳扎稳打才能取得最终胜利。”这是vivo打造历代产品积累起来的实战经验。</p><p>vivo方面曾表示，在过去的飞轮1.0时代，vivo通过为消费者提供产品与服务获得认可与增长，在被视为vivo质变元年的2021，vivo正式进入飞轮2.0时代，而这一时期的特征则是“产品是因，品牌是果。”vivo要做的是通过伟大的产品，建立消费者热爱的品牌。vivo的追求，不只限于打造优质产品，而是通过设计驱动、科技创新，创造“伟大的产品”，让消费者从选择到热爱，从满意到主动推荐。</p><p
